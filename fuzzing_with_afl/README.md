# Fuzzing with AFL++

## To install AFL++ 
```
git clone https://github.com/AFLplusplus/AFLplusplus.git
cd AFLplusplus
make
sudo make install
```

## Fuzzing with AFL++ in depth is available [here](https://aflplus.plus/docs/fuzzing_in_depth/)

***
All the folders have the scripts to start fuzzing process.
